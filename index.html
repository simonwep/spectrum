H

Toggle help screen (this one).

F

Toggle header.

R

Toggle between recording and using a file

CTRL / CMD+Up

Increase upper decibel limit.

CTRL / CMD+Down

Down upper decibel limit.

CTRL / CMD+SHIFT+Up

Increase lower decibel limit.

CTRL / CMD+SHIFT+Down

Decrease lower decibel limit.
